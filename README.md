# project_euler
Locked to private accordingly to challange's rules in order to not spoil the fun for others :)
Project euler is a site that offers a set of algorithmic challanges in form of mathematical problems for one to solve with a computer program. The problems start out easy and get progressively harder, so problem #1 is on average easier than
problem #1000. 
Current peak - level 3, [problem 54](euler_problem_54.py), "Poker Hands" - solved by 4% of the eulerians.
